XFLAG Alter3 Simulator について

パッケージ内容

Alter3Simulator.app
シミュレータソフトウェア本体です。
Alter3Simulator-Headless.app
シミュレータソフトウェア本体のヘッドレス版です。
TestClient.app
簡易テストクライアントです。

シミュレータのサーバ機能の操作

  1. シミュレータを起動すると自動的にサーバ機能も稼働開始します。サーバ機能が正常に稼働している場合、画面上部に server started と表示されます。
  2. クライアントソフトウェア側で、シミュレータに接続します。同一コンピュータ上で接続する場合は、localhost:11000 にTCP接続してください。
  3. サーバ機能を停止するには、左上のボタンをクリックしてサーバ機能を停止するか、ソフトウェアを終了します。
  4. 実行中の通信のログは、プラットフォームごとに以下の場所に出力されます。
    • Windows: C:\Users\<ユーザー名>\AppData\LocalLow\XFLAG\Alter3\Logs ※ドライブレターは環境により異なる場合があります。
    • macOS: /Users/<ユーザー名>/Library/Application Support/XFLAG/Alter3/Logs

シミュレータの画面操作

  • 画面上でスクロールやドラッグを行うと、カメラの向き変更やズームイン、ズームアウトができます。
    • 拡縮: マウスホイールのスクロールまたはトラックパッドの上下スワイプ
    • 移動: マウスの左ドラッグまたはトラックパッドの1本指ドラッグ
    • 回転: マウスの右ドラッグまたはトラックパッドの2本指ドラッグ
  • キーボードのFキーを押すことで、フェイスカメラの映像が画面右下に出現します。
  • キーボードのPキーを押すことで、カメラがシミュレータ起動時の状態に初期化されます。
  • キーボードのRキーを押すことで、ロボットのポーズがシミュレータ起動時の状態に初期化されます。
