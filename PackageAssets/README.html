Alter3 Simulatorについて

内容

  • Alter3Simulator.app
    シミュレータソフトウェア本体です。
  • TestClient.app
    簡易テストクライアントです。

シミュレータのサーバ機能の操作

  1. 画面左上のグレーの丸ボタンをクリックし、サーバを起動します。正常に動作を開始した場合、ボタンが緑色に発光し、server started と表示されます。
  2. クライアントソフトウェア側で、シミュレータに接続します。同一コンピュータ上で接続する場合は、localhost:11000にTCP接続してください。
  3. サーバを停止するには、画面左上のボタンを再度クリックしてサーバ機能を停止するか、ソフトウェアを終了します。
  4. 実行中の通信のログは、ウィンドウ下部に表示されているファイルパスに出力されます。

シミュレータの画面操作

  • 画面上でスクロールやドラッグを行うと、カメラの向き変更やズームイン、ズームアウトができます。
  • キーボードのLキーを3秒以上長押しすることで、ログウィンドウを表示することができます。内容はログファイルに出力されるものと同一です。
